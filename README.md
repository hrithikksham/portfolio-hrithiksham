A high-performance, cinematic developer portfolio designed to showcase engineering precision and design intuition. Built with the **Next.js App Router**, featuring scroll-driven animations, bento-grid layouts, and immersive interactions.

---

## ‚ö° Features

* **Cinematic Opening:** Parallax identity section with atmospheric lighting and particle effects.
* **Scroll-Driven Storytelling:** "Craft" manifesto that animates based on scroll velocity and position.
* **Bento Grid Architecture:** Apple-inspired project showcase with glassmorphism and hover physics.
* **Immersive Case Studies:** Full-screen, deep-dive modals for project details with zero layout shift.
* **High-Performance:** Optimized images (`next/image`), font loading, and strict type safety.
* **Cyber-Physical UI:** Custom cursors, neon glow effects, and sound-design inspired visuals.
